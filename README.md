# TypeScript 공부 기록을 위한 공간

## 목표
타입스크립트 기본 문법 익히기
완성된 코드를 보고 어떻게 실행되는지 이해할 수 있는 수준되기
타입스크립트 공부를 통해 자바스크립트도 이해하기

## 학습로그

[2025-12-17]

[1장] 타임스크립트 환경 세팅, [2장] 타임스크립트 프로젝트 생성과 관리

1. 환경 세팅
typescript를 공부하긴 위한 환경을 세팅함
tsconfig, package와 module 등 프로젝트를 만들 때 필요한 것들이 있다는 것을 깨달음
tsconfig에서 각 항목들이 뭘 의미하는지 공부함

2. 기본 구조를 익힘
export, import를 이용하여 소스코드를 모듈화(나누기)하는 법을 공부함
모듈이 모여서 하나의 프로그램이 된다는 것을 깨달음
타입스크립트 뿐 아니라 c언어 등 다른 언어들도 여러개의 소스코드들이 모여 하나의 프로그램이 된다는 것을 깨달음
