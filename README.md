## 객체와 타입

### 변수
- `let 변수: 타입 (= 초깃값)`이 기본이지만 초깃값에 따라 타입이 정해짐
- `const`는 상수로 초깃값 필요
- `any` 타입은 다 가능
- `undefined`는 타입이자 값이다
- 하위 타입의 변수에는 상위 타입의 값을 넣을 수 없다
- 템플릿 문자열은 f스트링 느낌으로 역따옴표와 `${}`를 사용  
```ts
let a = 10
let s = 'my age'
let result = `${s} is ${a}`
console.log(result) // my age is 10
